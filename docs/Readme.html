Configuring the bot

Go into the conf directory and edit "bot-config.xml".

Insert at a bare minimum the compulsory values for "jid" and "password".

Running the bot

java -jar CommandBot.jar

Developing on the bot

For embedding the bot, see the source in src/org/trypticon/commandbot/Main.java for sample usage.

For developing new commands, see examples/CalculatorCommand.java.

You can also consult the JavaDocs.
